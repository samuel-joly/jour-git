Commandes Description
git init Initialise Git in the current working directory
git log Show every commit on file made on this branch of the project
git remote Show a list of remote linked to this git directory
git remote add {remoteName} {url} Add a new remote named remoteName linked to GitHub URL
git add {filename} Add filename to your git for futur commits
git commit -m "My message" Allow to commit "My message" on the lastest modification added to git
git clone {url} {Directory} Download the selected git URL to selected Directory
git remote remove {remoteName} Remove remoteName from the linked remote
git push {remoteName} master Push your local git directory on GitHub
git revert {commitName} Undo and reset the file before the commitName
